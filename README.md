**📝 Coach CRM**

**Coach CRM** — это веб-приложение для репетиторов, разработанное для удобного управления расписанием занятий. Оно позволяет:

- Создавать **периодические** (еженедельные) и **разовые** занятия.
- **Отменять** конкретные занятия в серии, не удаляя всю серию.
- Визуально отличать отменённые занятия от активных.
- Сохранять данные в браузере (localStorage) для тестового режима.
- Переключаться между неделями и быстро возвращаться к текущей дате.

Проект построен на современных технологиях и ориентирован на простоту использования.

**🛠️ Технологии**

- **React** — библиотека для построения интерфейсов.
- **TypeScript** — типизированный JavaScript.
- **Zustand** — управление состоянием приложения.
- **Vite** — сборка и инструменты разработки.
- **CSS Modules** — изолированные стили для компонентов.
- **React Router** — навигация по страницам.

**🚀 Установка и запуск**

1. git clone https://github.com/ComShep/coach-crm.git
2. cd coach-crm
3. npm install
4. npm run build
5. npm run preview / npm run dev
