**📝 Notes App**

Приложение для создания, редактирования и хранения записей с облачным хранением данных.

**🌐 Демо**

https://comshep.github.io/note/

**✨ Возможности**

1. Просмотр, создание, редактирование и удаление записей;
2. Поиск по заголовку и содержимому записей;
3. Облачное хранение.

**🛠️ Технологии**

1. React 19;
2. TypeScript;
3. Firebase Realtime Database
4. Vite;
5. CSS Modules

**🚀 Устаовка и запуск**

1. git clone https://github.com/ComShep/note.git
2. cd note
3. npm install
4. npm run build
5. npm run preview / npm run dev

**📁 Структура**

src/
- api/          # API запросы
- contexts/     # React контексты
- hooks/        # Пользовательские хуки
- ui/           # UI компоненты
- types/        # TypeScript типы
